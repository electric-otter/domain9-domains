# Create a YAML file, looks a bit like this

```yaml
domain: example.tld
registrant:
  name: Your Name
  email: name@example.com
dns:
  - type: A
    value: my.ip.address
  - type: MX
    value: mail.example.com
expiration_date: When-It-Expires
```

Dont want your domain to expire? Just remove expiration_date
